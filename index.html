# 📦 품절 항목 코웍 프로그램 대시보드 무신사 채널 품절 항목을 엑셀 업로드만으로 자동 분석하는 웹 대시보드입니다. ## 🔗 바로가기 > 배포 후 이 부분을 실제 URL로 교체하세요 > `https://{깃허브유저명}.github.io/{레포이름}` --- ## ✅ 주요 기능 - **엑셀 업로드** → 현재고 0 자동 필터링 - **SET / 2PACK 파싱** → 단품 단위로 자동 분해 및 합산 - **2가지 뷰** → 색상·사이즈별 상세 / 상품별 합산 - **검색 + 필터** → 상품명 검색, 단품/SET/2PACK/복합 필터 - **컬럼 정렬** → 주문수량 기준 내림차순 등 ## 📂 지원 파일 형식 - 무신사 2주 주문 데이터 (`.xlsx`, `.xls`) - 필수 컬럼: `상품명`, `옵션`, `현재수량`, `수량` ## 🚀 GitHub Pages 배포 방법 1. 이 레포를 **GitHub에 업로드** 2. `Settings` → `Pages` → Source: `Deploy from a branch` 3. Branch: `main` / `(root)` 선택 후 **Save** 4. 약 1~2분 후 URL 생성 완료 --- ## 🛠 기술 스택 - Vanilla HTML / CSS / JavaScript - [SheetJS (xlsx)](https://sheetjs.com/) — 엑셀 파싱 - Google Fonts (Pretendard, DM Mono) - 외부 서버 불필요 — 완전 클라이언트 사이드
